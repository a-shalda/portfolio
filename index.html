PORTFOLIO

Bricks eCommerce website

Customers who shop for building materials do not usually buy online, partially because they need help calculating the needed volume. On this website I implemented pre-calculated options, which help to see the subtotal, number of packs and pieces, weight and the number of pallets before adding the product to the cart. Plus, customers can see prices for a square meter and for a running (linear) meter for respective products.

{Tech stack: HTML, CSS, SaSS, JavaScript}
